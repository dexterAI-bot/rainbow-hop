# Rainbow Hop (קפוץ קשת)

A colorful, kid-friendly one-tap game written in HTML/JS. The heroine is always ready for a jump; tap the screen, climb between sparkly platforms, and collect stars while cheerful Hebrew text and bubbly beeps guide the experience.

## Features

- Hebrew UI text and instructions (even on the buttons).
- Canvas-powered jumper that resets as soon as the star count updates.
- Soft gradients, pastel platforms, and a friendly ribbon emoji as the avatar.
- Built-in beeps using the Web Audio API (muted by default with a toggle).
- Mobile-friendly layout optimized for phone viewing.

## Launching
1. Drop the folder onto any static hosting (GitHub Pages, dashboard viewer, etc.).
2. Open `index.html` in a browser on your phone.
3. Tap `התחל` (start) and then tap the canvas to keep hopping.

No build tools or dependencies required—just plain HTML/JS/CSS.
